No celular você escolhe qual câmera usar. No PC normalmente só há uma webcam.

Em alguns telemóveis (sobretudo Android com Chrome) o zoom real da câmera é usado; no iPhone/Safari costuma ser simulação no ecrã e no vídeo (corte central / barras). Não troca de lente ultra‑larga sozinho.

Opcional: se escolher um ficheiro, só esse áudio entra no vídeo. O microfone não é captado.

Duração final estimada: 10s

Se o MK360 está noutro domínio (ex.: *.github.io), coloque aqui o endereço onde o Node está na Internet (HTTPS de preferência). Esse host responde a /api/upload e /api/drive-upload com CORS.

Após cada vídeo finalizado, o ficheiro é enviado automaticamente para a pasta MK360 no Drive (configurada no google-apps-script-drive.gs).

Cole a URL /exec e salve. O envio usa o proxy /api/drive-upload no URL do servidor Node acima (local ou público). Em site estático, esse URL tem de ser o mesmo domínio onde corre o server.js. O script no Drive define o ficheiro como “quem tem o link” para o QR abrir sem login. Copie alterações do google-apps-script-drive.gs para o Apps Script e publique uma nova versão.
